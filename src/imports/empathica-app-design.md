Design a clean, modern mobile app UI called Empathica.

Empathica is a futuristic wellness-focused social perception app that helps users understand social signals, emotional atmospheres, and interpersonal dynamics during conversations. The design should feel like Apple-level minimalism, friendly and approachable, with optional light mode and dark mode.

The app analyzes real-time and past social interactions and translates subtle signals like tone, body language, facial cues, and emotional atmosphere into easy-to-understand metrics that promote self-awareness and emotional wellness.

The background color of the interface dynamically changes based on the detected emotional atmosphere, using a color system such as:

Blue = Calm (0–20)

Green = Neutral (21–40)

Orange = Energetic / Stimulating (41–60)

Red = Tense (61–80)

Deep Red = Highly stressful (81–100)

Allow gradient transitions between colors to visualize shifts in emotional atmosphere. Users can customize these color mappings in settings.

Create a mobile UI design with the following key screens and components:

Home Dashboard

Shows the current emotional atmosphere score (0–100) with a soft gradient background color reflecting the environment.

Displays a large Social Emotional Battery meter indicating the user's current social energy level.

Quick summary cards for:

Current emotional atmosphere

Social energy remaining

Active interaction insights

Minimalistic icons and rounded cards.

Real-Time Interaction Screen

Designed for use during conversations.

Displays live metrics such as:

“Do they like me back?” attraction probability (percentage gauge)

Lie detection probability (percentage indicator)

Social engagement level

Show subtle waveform or signal indicators to represent live social cues being analyzed.

Interaction Analysis Screen

Post-conversation analysis.

Timeline of the interaction with emotional fluctuations.

Attraction probability summary.

Lie detection highlights.

Social cue breakdown (eye contact, tone shifts, emotional tension).

Interaction History

List of past interactions with timestamps.

Each interaction card shows:

atmosphere score

social battery impact

key insights.

Include visual indicators like small colored emotion dots.

Emotional Atmosphere Map

Visual dashboard showing the emotional atmosphere score with gradient visuals and an explanatory scale from calm to highly stressful.

Encourage reflection and emotional awareness.

Settings Screen

Users can customize emotion-to-color mappings.

Toggle light mode / dark mode.

Privacy settings.

Option for smart glasses integration for future real-time body language detection.

Design style guidelines:

Clean Apple-like minimal UI

Rounded cards and soft shadows

Friendly typography

Subtle gradients

Smooth emotional color transitions

Overall tone should feel supportive, insightful, and empowering, helping users better understand social interactions and their own emotional state.
